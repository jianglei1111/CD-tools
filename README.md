# CDcode

CDcode 是一组面向 Codex 的本地中转工具集合，用来把不同入口的消息、文件和图片能力转接到 Codex 工作流里。

## 工具列表

### Chedan Image2 Skill

`chedan-image2-skill` 是车单 image2 图片生成与编辑技能包。安装后，Codex 可以通过固定的 image2 通道生成图片、编辑图片，并把结果保存到本地工作目录。

### QQ-Codex

`qq-codex` 是 QQ 到 Codex 的本地桥接器。它通过 QQ 开放平台 Bot API 接收 QQ 私聊或群聊消息，并把消息转发给本地 Codex 执行。

### Weixin-Codex Bridge

`weixin-codex-bridge` 是微信到 Codex 的本地桥接器。它通过微信 iLink Bot API 接收微信消息、图片和文件，并把会话接入本地 Codex。
